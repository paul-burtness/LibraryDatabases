The purpose of this app is to display the list of Anoka County Library online databases in the context of the Library's page on the public website, www.anokacounty.us

The list of databases is maintained in the Excel spreadsheet - DatabaseList.xlsx

The Excel Spreadsheet is converted to XML, following ideas at http://www.nextofwindows.com/how-to-convert-excel-spreadsheet-to-json

The valid XML file is converted to JSON at http://codebeautify.org/xmltojson

The resulting JSON file is validated at http://jsonlint.com/
